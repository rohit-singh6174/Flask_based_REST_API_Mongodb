Flask REST API CRUD Operaion using Mongo Db
1. GET /users - Returns a list of all users
2. GET /users/ - Returns the user with the specified ID.
3. POST /users - Creates a new user with the specified data.
4. PUT /users/ - Updates the user with the specified ID with the new data
5. DELETE /users/ - Deletes the user with the specified ID
